## Goal

Reshape the orders module of `apps/retail-microservice/src/` into the
per-module hexagonal layout. Orders is more complex than products:
the `ConfirmOrderUseCase` orchestrates a cross-service call into
inventory's `OrderConfirm` handler, so this task also formalizes the
inventory-gateway port that orders depends on.
